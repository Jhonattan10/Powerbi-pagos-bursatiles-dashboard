Proyecto-powerbi-pagos-bursatiles
Análisis de pagos bursátiles utilizando Power BI (DAX, modelado en estrella y dashboards interactivos).

Proyecto de análisis de pagos y conciliación de transacciones bursátiles mediante Power BI.

Objetivo  
Analizar el comportamiento de los pagos bursátiles y el estado de conciliación de las transacciones mediante dashboards interactivos, enfocados en montos procesados, volumen de transacciones, métodos de pago, clientes y control operativo.

Tecnologías utilizadas  
Power BI  
DAX  
Power Query  
Excel  

Modelo de datos  
El modelo de datos está diseñado bajo un esquema estrella y está compuesto por las siguientes tablas principales:

Transacciones  
Clientes  
Servicios  
Métodos de Pago  
Fechas  

Métricas implementadas  
Monto Total Procesado  
Monto Total  
Monto Conciliado  
Monto Pendiente  
Monto Observado  
Total de Transacciones  
Ticket Promedio  
% Conciliado  
% Pendiente  
% Observado  
% del Monto por Método de Pago  

Descripción  
El proyecto simula un entorno de pagos bursátiles realista, con transacciones procesadas en distintas monedas y métodos de pago.  
El dashboard está diseñado para brindar una visión ejecutiva y operativa del proceso de pagos, facilitando el monitoreo del desempeño y la identificación de transacciones que requieren seguimiento.
