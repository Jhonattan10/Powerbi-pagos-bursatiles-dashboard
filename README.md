# Proyecto Power BI – Pagos Bursátiles

Análisis de pagos y conciliación de transacciones bursátiles utilizando **Power BI**, con métricas en **DAX**, modelado en estrella y dashboards interactivos.

---

## Objetivo

Analizar el comportamiento de los pagos bursátiles y el estado de conciliación de las transacciones mediante dashboards interactivos, enfocados en montos procesados, volumen de transacciones, métodos de pago, clientes y control operativo.

---

## Tecnologías utilizadas

- Power BI
- DAX
- Power Query
- Excel

---

## Modelo de datos

El modelo de datos está diseñado bajo un **esquema estrella**, compuesto por:

- **Transacciones** (tabla de hechos)
- **Clientes**
- **Servicios**
- **Métodos de Pago**
- **Fechas**

---

## Métricas creadas (DAX)

- Monto Total  
- Monto Total Procesado  
- Monto Conciliado  
- Monto Observado  
- Monto Pendiente  
- Ticket Promedio  
- Total de Transacciones  
- % Conciliado  
- % Monto por Método de Pago  
- % Observado  
- % Pendiente  

---

## Descripción

El proyecto simula un entorno realista de **servicios de pago bursátiles**, con datos estructurados para análisis financiero y operativo.
